# Buto-Plugin-IconsOcticons
Svg icons.

## Icon
Showing an alert icon.
```
type: widget
data:
  plugin: 'icons/octicons'
  method: svg
  data:
    name: alert
```

### Attribute
One could add attribute param.
```
type: widget
data:
  plugin: 'icons/octicons'
  method: svg
  data:
    name: alert
    attribute:
      id: _my_svg_icon_
```

## List of all icons
List all icons with names.
```
type: widget
data:
  plugin: 'icons/octicons'
  method: list
```
